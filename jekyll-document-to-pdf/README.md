## 将 jekyll 文档转成 pdf
---

### 运行
---

```
python3 jekyll_spide.py
```

### 数据来源
---

 [jekyll document](https://jekyllrb.com/docs/home/)


### 库
---

 - pip install **requests**

 - pip install **beautifulsoup**

 - pip install **pdfkit**


安装 **wkhtmltopdf**

 - $ sudo apt-get install **wkhtmltopdf**  # ubuntu

 - $ sudo yum intsall **wkhtmltopdf**      # centos
